❓ 使用說明

快速開始

  1. 選擇或建立一個「專案」。
  2. 選擇或建立一支「SQL」腳本。
  3. 在 SQL 編輯器輸入內容。
  4. 點選「保存版本」，填寫版本標籤與作者。
  5. 需要單檔時，可點選「下載 SQL」下載目前編輯器內容。
  6. 之後可從版本列表查看、載入或比較版本。

💾 保存版本

SQL 修改到一個段落後，點選「保存版本」。版本標籤建議寫清楚，例如「新增索引」或「上線前版本」。

🔍 比較版本

點選「比較模式」，在左右兩側各選一個版本，就能查看新增、刪除與未變更內容。

⬇️ 下載 SQL

「下載 SQL」會把目前編輯器中的內容下載為 .sql 檔案，適合交付單支腳本；不會保存成版本。

📤 導出與導入

「導出」可下載單一專案版本歷史；「導入」可把 JSON 檔案匯回系統。遇到衝突時，不確定可選「合併為新版本」。

💾 完整備份

完整備份會保存所有專案、SQL 腳本與版本。清理瀏覽器資料、換電腦或大量修改前，建議先備份。

重要提醒

資料主要保存在目前瀏覽器本機。清除網站資料、換瀏覽器或換電腦時，原本資料不會自動出現，請使用完整備份與完整還原搬移資料。
